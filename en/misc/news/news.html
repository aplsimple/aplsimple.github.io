10 Sep 2019

A mirror at chiselapp.com was created. Some repositories were created at GitHub and Chisel.

A page at wiki.tcl-lang.org was started. A few words about Fossil and Chisel.

e_menu 1.30:
- yesnocancel dialog for Q to be used in a1= argument
- multiple commands separated with ";"in a1=
- file attributes of menus are kept at choosing items
- project name underlined (for spec.characters)
- %F_ wildcard = %F underlined
- fI=, bI= arguments for active item's colors
- local Tcl help's file name normalized
- local Tcl help's contents displayed if no page found
- .mnu changed


25 Aug 2019

Moving from aplsimple.bitbucket.io to aplsimple.github.io. Less letters, more sense.

e_menu 1.28:
- exec -ignorestderr when running Linux terminal
- "# ARGS1.." works as "#ARGS1.." for in-text arguments
- possible issue with 'workdir' fixed
- few .mnu changed
- pave updated

pave:
- in editor, Alt+Up/Down to move lines up/down
- undo is enabled by default for text widget


4 Aug 2019

e_menu 1.27:
1. ln=, cn= arguments, to read a word under caret if no selection
2. yn= argument, to confirm exit at Escape pressing
3. in=1.0 in [OPTIONS] section, to save & restore last run position
