О чем это

Здесь плоды моих потуг в русском сегменте, так сказать. Есть еще ссылка на английский сегмент, справа вверху.

Заметьте: при скачивании файлов вы можете получить антивирусное предупреждение - не пугайтесь, оно стандартно для скачиваемых ".exe" файлов.

Все программы проверены антивирусом. Мин нет.


FVords

Программа FVords - это бесплатная "языковая обучалка", заточенная под изучение английского русскоязычными пользователями. Применение FVords для других языков ограничено как по объему баз, так и по доступности некоторых режимов обучения. Программа работает в Microsoft Windows, но с некоторыми ограничениями может работать и под Wine в Linux'е.

Программа FVords + англ.база

Полный (и объемный, ~75Mb) инсталлятор FVords и английской базы скачивается по ссылке:
     FVords+English-base-1.13.30.exe

Обновление программы FVords (без базы) содержит обновленную версию FVords v1.13.31, его рекомендуется накатить на предыдущую инсталляцию.

После установки FVords рекомендуется также скачать и установить обновление английской базы FVords (fv_engl.exe).

Английская база FVords

Английская база FVords (без книг и mp3) скачивается по ссылке:
     fv_engl.exe

Немецкая база FVords

Немецкая база FVords скачивается по ссылке:
     fv_germ.exe

Прочие базы FVords

База эсперанто FVords скачивается по ссылке:
     fv_esper.exe


Другие программы

Остальные программы не столь объемны, как FVords, но тоже заслуживают внимания, надеюсь.

APLinkee

APLinkee - архивариус ссылок (для вызова программ, документов, страниц Интернета и т.п.) плюс небольшой органайзер. Говорят, очень функционально.
Cкачивается по ссылке:
     arlsetup138.exe

FVDict

FVDict - англо-русско-английский cловарь. ~87000 слов в англо-русской части и ~45000 в русско-английской.
Доступен по ссылке:
    fvdictsetup.exe

Reportf

Reportf - генератор отчетов в форматах RTF, TXT, HTML. Есть полный, "легкий" варианты и вариант для Lazarus IDE.
Cкачивается по ссылкам:
     reportf.zip
     reportfl.zip
     reportfl_laz.rar

ProvSlav

ProvSlav - прогноз демографической ситуации в России.
Cкачивается по ссылке:
     pssetup.exe

ORDic

ORDic - русский орфографический словарик.
Cкачивается по ссылке:
     ORDsetup.exe

lib_ru

lib_ru - обработчик текстов для чтения в электронных читалках (по материалам lib.ru).
Cкачивается по ссылке:
     lib_ru.rar

replstr

replstr - пакетная замена строк в текстовых файлах.
Cкачивается по ссылке:
     replstr.rar

caja-nautilus scripts

caja-nautilus scripts - сценарии для файловых менеджеров Linux.
Cкачиваются по ссылке:
     caja-nautilus-scripts.rar
